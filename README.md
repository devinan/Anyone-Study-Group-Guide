> 입문자의 대상이므로 처음에는 기본적인 html, css, javascript 위주로 진행한다.
> 대부분의 유료나 무료강의 등 좋은 자료가 많이 접근할수 있기 때문에 스스로 듣는다는 가정하에 스터디에서는 이론에 대한 부분을 짧게 진행을 하고 첨삭 및 실제 개인 프로젝트 지도 수업 중심으로 목적을 맞춘다.
